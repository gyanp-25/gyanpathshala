# GitHub Pages Teacher Portfolio Setup

Welcome to your GitHub Pages teacher portfolio! This guide will help you set up and customize your portfolio website.

## Prerequisites
- A GitHub account
- Basic understanding of Git and GitHub
- Familiarity with Markdown (optional, but helpful)

## Setting Up Your Portfolio
1. **Fork the Repository**: Click on the ‘Fork’ button to create your own copy of the repository.
2. **Clone the Repository**: Use the command below to clone your new repository to your local machine:
   ```bash
   git clone https://github.com/gyanp-25/gyanpathshala.git
   ```
3. **Navigate to the Repository**:
   ```bash
   cd gyanpathshala
   ```
4. **Launch the Local Server**: You can use GitHub Pages to host your site. Simply navigate to the repository settings and enable GitHub Pages under the ‘Pages’ section.

## Customizing Your Portfolio
- **Edit the `_config.yml` File**: Update the title, description, and other settings specific to your portfolio.
- **Add Your Information**: Include your name, bio, educational background, and any other information you want to showcase.
- **Upload Your Work**: Create new folders for different subjects or projects and upload your work (documents, pictures, etc.).

## Deploying Your Site
After customizing, make sure to commit your changes:
```bash
git add .
git commit -m "Set up initial portfolio"
git push origin main
```
Your site should be live in a few minutes at `https://gyanp-25.github.io/gyanpathshala`.

## Conclusion
Congratulations! You’ve successfully set up your GitHub Pages teacher portfolio. For further customization, refer to the [GitHub Pages documentation](https://pages.github.com/).
